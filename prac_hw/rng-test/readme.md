# Валидация генераторов случайных чисел

Требуется для фиксированного генератора случайных чисел:
1. Проверить соответствие выходных данных генератора нормальному распределению $N(0,1)$ с помощью критерия Колмогорова-Смирнова (KS).
2. Протестировать битовую последовательность, вырабатываемую генератором, на случайность с помощью трёх тестов: тест Фурье, автокорреляции и тест на промежутки (gaps).
3. Для каждой проверки применить **двухуровневое тестирование**: собрать p‑значения для множества независимых последовательностей, а затем проверить равномерность этих p‑значений с помощью критерия KS.

## Библиотеки
- В качестве генератора нормального распределения используется `numpy.random.normal(0, 1)`.
- Для проверки битовых последовательностей используется `numpy.random.MT19937` через `bit_generator.random_raw()`: из сырых 32-битных слов генератора строится поток битов.
- Дополнительно реализован пример «плохого» генератора для демонстрации ограниченности отдельных статистических тестов.

## Двухуровневое тестирование
Процедура для каждого теста:
1. Генерируется N независимых последовательностей (или выборок) заданного размера.
2. Для каждой последовательности вычисляется тестовая статистика и соответствующее p‑значение в предположении, что нулевая гипотеза ($H_0$) верна.
3. Полученные p‑значения $(p_1, \dots, p_N)$ собираются в массив.
4. С помощью критерия Колмогорова-Смирнова проверяется гипотеза о том, что эти p‑значения равномерно распределены на отрезке [0,1].
5. Если итоговое p‑значение (KS‑теста на равномерность) больше 0.05, считается, что генератор успешно прошёл тест (на уровне значимости 5%).

## Тесты

### Критерий Колмогорова-Смирнова
> $H_0$: выборка происходит из стандартного нормального распределения N(0, 1).

В качестве статистики используется максимальное расстояние между эмпирической функцией распределения и теоретической функцией нормального распределения. Одновыборочный критерий реализован функцией `scipy.stats.kstest`.

### Тест Фурье
> **$H_0$**: битовая последовательность случайна (спектр ДПФ ведёт себя как ожидается).

Алгоритм:
  1. Биты преобразуются в значения +1/-1
  2. Вычисляется дискретное преобразование Фурье (через БПФ)
  3. Подсчитывается количество $C$ частотных компонент, модуль которых меньше порога $h = \sqrt{2.9957n}$
  4. При справедливости $H_0$ величина $C$ приближённо нормальна с $E = 0.95n$ и $D = 0.05 \cdot 0.95n$
  5. Вычисляется нормированное отклонение $z = \frac{(C - 0.95n)} {\sqrt{0.05 \cdot 0.95n}}$ и двустороннее p‑значение

### Тест автокорреляции
> $H_0$: биты независимы и равновероятны.

Алгоритм:
  1. Вычисляется сумма $A = \sum_{i=0}^{n-2} (b_i \oplus b_{i+1})$
  2. При $H_0$ величина $A$ имеет биномиальное распределение $\mathrm{Bin}(n-1, 0.5)$
  3. Двустороннее p‑значение получается из биномиального распределения

### Тест на промежутки (Gaps)
> $H_0$: расстояния между последовательными единицами подчиняются геометрическому распределению с вероятностью успеха $p = 0.5$.

Алгоритм:
  1. Находятся позиции всех единиц
  2. Вычисляются промежутки (количество нулей между соседними единицами)
  3. Промежутки группируются в категории: 0, 1, 2, 3, 4, и $\ge 5$
  4. Наблюдаемые частоты сравниваются с ожидаемыми по геометрическому распределению с помощью критерия $\chi^2$
  5. Возвращается p‑значение $\chi^2$-теста

## Параметры
- Для проверки нормальности создаётся $N = 100$ выборок, каждая объёмом 1000.
- Для битовых тестов используется $N = 50$ последовательностей, каждая длиной 10000 (кроме теста Фурье, где длина 1024, для удобства БПФ).
- Уровень значимости финального KS‑теста: $\alpha = 0.05$

### Плохой генератор
В демонстрационном плохом генераторе каждый 32-й бит принудительно обнуляется. Это создаёт структурный дефект в потоке, но не обязательно обнаруживается каждым отдельным тестом.

## Результаты

### Проверка нормальности
| Тест               | Двухуровневое p |
|--------------------|-----------------------------------------------|
| KS  | 0.4530                                 |


### Тестирование битовых последовательностей
| Тест               | Двухуровневое p |
|--------------------|-----------------------------------------------|
| Фурье              | 0.0921   |
| Автокорреляция (lag 1) | 0.2951  |
| Промежутки (gaps)  | 0.5848   |

Все двухуровневые p‑значения больше 0.05, следовательно, поток битов `MT19937 bit generator` не демонстрирует отклонений от случайности в применённых тестах.

### Плохой генератор
| Тест               | Двухуровневое p |
|--------------------|-----------------------------------------------|
| Фурье              | 0.7574   |
| Автокорреляция (lag 1) | 0.0975 |
| Промежутки (gaps)  | 0.0000   |

Плохой генератор проходит тест Фурье и автокорреляции при уровне значимости 5%, но проваливает тест на промежутки. Это показывает, что нескольких проверок может быть недостаточно, так как генератор может выглядеть хорошим по части статистик и иметь явный дефект по другим.

## Заключение

Исследуемые генераторы (`numpy.random.normal` и поток битов `MT19937 bit generator`) успешно прошли применённые тесты в рамках двухуровневой процедуры. Все итоговые p‑значения критерия KS на равномерность p‑значений оказались больше 0.05. Также продемонстрирован контрпример плохого генератора, который проходит некоторые тесты, но проваливает другие, что подтверждает необходимость набора независимых статистических проверок.
