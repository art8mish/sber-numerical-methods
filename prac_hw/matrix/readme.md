# Оптимизация матричного умножения

Требуется оптимизировать вычисление `C = A * B` (без BLAS) и сравнить разные оптимизации.

## Реализации матричного умножения

1. **Наивная**: тройной цикл `i, j, p`, в `B` проход по столбцу.

2. **Блочная**: блочный порядок `jj, kk, ii` (эквивалентно `N, K, M` по размерностям блоков) и внутреннее накопление `C[i, j] += A[i, p] * B[p, j]` в порядке `i, p, j`.

3. **Блочная + SIMD (AVX2/FMA), `simd_ipj`**: порядок индексов как в блочной скалярной версии (`i, p, j`), векторизация по `j`, обновление `C` на каждом `p`.

4. **Блочная + SIMD (AVX2/FMA), `simd`**:  с накоплением по `p` в регистрах с аккумуляцией сумм в `C`.

## Сборка и запуск

```shell
cd prac_hw/matrix
make
./matrix
```

- Строка сборки: `g++ -O3 -Wall -Wextra -std=c++17 -march=native -o matrix matrix.cpp`
