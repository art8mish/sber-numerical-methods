# Оптимизация матричного умножения

Требуется оптимизировать вычисление `C = A * B` (без BLAS) и сравнить разные оптимизации.

## Реализации матричного умножения

1. **Наивная**: тройной цикл `i, j, p`, в `B` проход по столбцу.

2. **Блочная**: предвычисляется `B^T`, проход для внешнего цикла производится по индексам `jj ii kk`, что позволяет кешировать слои матрицы `B` в `L3`, а блоки `A` в `L2`, для внутренного по `i j p`, что позволяет минимизировать обращение к `C` для минимазации промахов.

3. **Блочная + SIMD (AVX2 и FMA)**: к блочной реализации добавлена SIMD свёртка по `p` во внутреннем цикле

## Сборка и запуск

```shell
cd prac_hw/matrix
make
./matrix
```
