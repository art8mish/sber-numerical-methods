
# Потеря точности float[32|64]

Требуется исследовать проблему потери точности, реализовав три метода для вычисления оценки дисперсии в float[32|64] с использованием как минимум 3 выборок:
- 1000 чисел из нормального распределения (среднее = 1, среднеквадратическое отклонение = 1)
- 1000 чисел из нормального распределения (среднее = 10, среднеквадратическое отклонение = 0.1)
- 1000 чисел из нормального распределения (среднее = 100, среднеквадратическое отклонение = 0.01)

## Вычисление дисперсии $DX$

При использовании чисел с плавающей точкой выбранный алгоритм критически влияет на точность результата.

## 1. "Быстрый" алгоритм
Данный алгоритм основан на определении:
```math
DX = \frac{1}{n} \sum_{i=1}^{n} x_i^2 - \left( \frac{1}{n} \sum_{i=1}^{n} x_i \right)^2
```
При близких значениях уменьшаемого и вычитаемого происходит потеря значимости, вплоть до отрицательности результата.

## 2. Двухпроходный алгоритм
В основе алгоритма лежит поочередное вычисления среднего $M$ и сумма квадратов отклонений $DX$:
```math
M = \frac{1}{n} \sum_{i=1}^n x_i \\
DX = \frac{1}{n} \sum_{i=1}^{n} (x_i - M)^2
```
Данный алгоритм, по сравнению с предыдущим, более устойчив, но требует два прохода по данным.

## 3. Однопроходной алгоритм
Алгоритм позволяет вычислять дисперсию за один проход с высокой точностью. В основе работы лежит обновление среднего $M_n$ и суммы квадратов $DX_n$ для каждого нового значения $x_n$:

```math
M_{n+1} = M_{n} + \frac{x_{n+1} - M_{n}}{n + 1} \\
```
```math
DX_{n+1} = DX_{n} + \frac{(x_{n+1} - M_{n})(x_{n+1} - M_{n+1}) - DX_n}{n + 1}
```

###  Ошибка

Ошибка считалась относительно порядка дисперсии. Для `double` разницы в методах нет. Для `float` видно проявление критической потери точности для *быстрого* алгоритма - отрицательное значение дисперсии для последней выборки и ошибка в 200%. Среди алгоритмов заметна теннденция: *двухпроходной* алгоритм даёт лучший результат, *быстрый* в ннекоторых случаях выдаёт серьёзную ошибку из-за вычитания близких величин, *однопроходной* алгоритм сравним с *двухпроходным* по точности, но имеет явное преимущество по времени.
